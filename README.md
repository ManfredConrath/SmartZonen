# SmartZonen

Digitale læringsværktøjer til indskolingen.

## Opsætning på GitHub Pages

1. Upload alle filer til dit GitHub-repository
2. Gå til Settings → Pages
3. Vælg branch: `main`, mappe: `/` (root)
4. Din side er tilgængelig på: `https://<brugernavn>.github.io/<repository-navn>/`

## Tilføj bannerbillede og logo

- Gem dit bannerbillede som `assets/banner.jpg`
- Gem dit logo som `assets/logo.png`

## Tilknyt domænet smartzonen.dk

1. Tilføj en fil kaldet `CNAME` med indholdet: `smartzonen.dk`
2. Opdater DNS hos din domæneudbyder:
   - A-records til GitHub's IP-adresser
   - Eller CNAME-record der peger på `<brugernavn>.github.io`
